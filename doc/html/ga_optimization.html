Feature weighting is a generalizationnof feature selection with real-valued weights between [0,1] instead of thenbinary values 0/1.nnDue to the fact that feature selection and weighting are NP-hard optimizationnproblems, a complete and correct solution is not easy to determine. One waynto get an approximate solution is by using genetic algorithms. Even thoughnfeature weighting is a generalization of feature selection and should thusnbe able to produce better classification results, practice has shown thatnfeature selection produces better recognition rates in most cases. It isnthus recommended to use feature selection rather than feature weightingn[Bolten2012]_.nnGenetic algorithms, proposed by Holland, are adaptive problemnsolvers that find solutions based on the mechanics of natural selection andnnatural genetics. For more information, see a genetic algorithmsntext such as [Holland1975]_nnHow the optimization can be used after training the classifier is described in the `training\ntutorial`__.nn.. __: training_tutorial.htmlnnUsagen-----nnThere are two ways to use the optimization:nn- Graphical User Interface: `Biollante`__nn- Python `Script API`__nnThese two approaches are equivalent, although due to the required runtimenthe script API is recommended.nn.. __: #user-interface-biollanten.. __: #script-usagennnUser Interface Biollanten''''''''''''''''''''''''''nn "Biollante is the result of an experiment in geneticn engineering.
