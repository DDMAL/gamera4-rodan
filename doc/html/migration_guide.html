Sincenit would be impossible to change the order of the constructor'snarguments without breaking backward imcompatibility, a new type hasnbeen introduced, Dim, which is identical to Dimensions in allnrespects except its constructor is Dim(ncols, nrows). The standard Gamera Point storesncoordinates as unsigned (positive) integers, and doesn't have anynarithmetic operators. For this reason, FloatPoint is highlynrecommended for any analyses that require precision and flexibility.n``Point`` is kept around for backward compatibility, and because it isna more natural way to refer to physical pixels, as opposed to logicalncoordinates.nnThere are, however, implicit conversions (in Python only) between thentwo types, so a FloatPoint can be used in place of Point wherenit makes sense. Care should be taken to ensure that negative valuesnare never used to reference image pixels.
